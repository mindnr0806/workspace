JSP (Java Server Page)

JSP란, HTML 내에 자바코드를 쓸 수 있는 자바 언어이다.
Servlet에서 응답화면을 구현했던 복잡함을 간단하게 하는 기능
=> 즉, JSP의 가장 큰 장점은 Servlet에서는 비즈니스 로직에만 집중하고
프리젠테이션 로직(화면 구현)은 JSP에서 집중하도록 분리할 수 있는 것

* JSP Element 표현법 (JSP 문법)

(1) JSP 스크립트 원소

JSP페이지에서 자바코드를 기술할 수 있는 문법(기능)

  1. 선언문 : <%! 자바코드 %>
    멤버변수와 메소드를 선언하기 위해서 사용
  2. 스크립트릿 : <% 자바코드 %>
    JSP에서 자바코드를 사용하기 위한 기본적인 영역
  3. 표현식(출력식) : <%= 자바코드 %>
    자바코드의 값을 화면상에 출력하기 위해 사용
view detail

(2) 지시어

JSP 페이지 전체에 영향을 미치는 설정(정보)을 작성할 때 사용
<%@ page|include|taglib 속성="속성값" 속성="속성값" %>

  1. page 지시어 : 현재 jsp페이지를 처리하는데 필요한 속성들을 작성하는 부분
    • language : 사용할 언어 유형 지정
    • contentType : 해당 페이지의 형식, 문자셋 지정
    • pageEncoding : 해당 페이지 내에 기록된 자바코드의 인코딩 방식
    • import : 자바의 import와 같은 의미
    • errorPage : 현재 페이지에서 오류가 발생했을 경우 보여줄 에러페이지 경로 지정
    view detail
  2. include 지시어 : 현재 JSP 페이지 내에 또 다른 jsp 페이지를 포함시키고자 할 때 사용하는 지시어
    view detail
  3. taglib 지시어 : JSP 기능을 확장시킬 수 있는 추가적인 라이브러리를 등록할 때 사용
