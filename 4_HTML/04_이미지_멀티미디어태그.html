이미지 태그

* src 속성

* alt 속성

- 이미지 경로가 잘못되었거나 제대로 표시하지 못하는 경우 대체할 텍스트
- 시각 장애인들을 위한 스크린리더에서 이미지에 대한 설명을 읽어줄 수 있음


* width, height 속성

이미지의 가로(width), 세로(height) 길이 조정 가능
고정길이(px), 가변길이(%) 지정 => 기본값 : px

고정 길이(px) - 기본값

화면 사이즈가 변경되어도 이미지의 크기는 고정(변동 없음)

가변 길이(%)

화면 사이즈가 변경되거나 부모 요소 사이즈에 따라 이미지 크기 변경
