문단을 구분하는 태그 : p, pre

문단을 나누는 태그로 p태그와 pre태그가 있다. p 태그 내에서는 줄바꿈처리를 하고자 한다면 별도의 태그를 사용해야 함
**줄바꿈 처리**
그리고 공백은 한 개의 공백만 표시하므로 별도의 기호 문구를 사용해야 함
-> 기호 문구를 사용하지 않은 경우 : @ @
-> 기호 문구를 사용한 경우 : @       @

그 외의 글자 관련 태그들

일반적인 글자 표현하기 : 태그 없이도 글자 표시 가능!

strong : 글자를 굵게 표시하는 태그 (+ 스크린 리더)

em : 글자를 기울여서 표시하는 태그

mark : 형광펜 효과와 함께 표시하는 태그

u : 글자에 밑줄을 표시하는 태그

s : 글자에 취소선을 표시하는 태그

small : 글자를 작게 표시하는 태그

첨자 표시 태그
글자에 아래첨자를 표시하는 태그와 윗 첨자를 표시하는 태그
ex) a2 + b2 = c2
ex) H2O

태그 중첩 사용
굵은 글자, 기울여서 표시
취소선, 형광펜 표시
