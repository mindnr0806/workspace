  • 목록이다!!!!

    • ul : 순서 없는 목록 태그

    • HTML
    • CSS
    • JavaScript
    • jQuery
    • HTML 태그
      • 글자 관련 태그 : p, pre
      • 목록 관련 태그 : ul, li
    • CSS
    • JavaScript
    • jQuery

    ol : 순서 있는 목록 태그

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    숫자로 표시, 7부터 시작, 역순으로

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    영문으로 표시, 7번째 알파벳부터 시작

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    아라비아 숫자로 표시, 3번째부터 시작

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    dl : 설명 관련 목록 태그 (주로 용어 정의 시 사용)

    제목 입력 태그
    설명 입력 태그
    프론트엔드 영역
    HTML, CSS, JavaScript 등의 언어를 사용하여 웹 브라우저 화면을 구성
    백엔드 영역
    Java, DB 등을 사용하여 사용자의 요청에 대한 처리 후 응답하여 서비스를 제공
